# GearGuard

Repository: https://github.com/showman-patil/Gear-guard.git

Clone:

    git clone https://github.com/showman-patil/Gear-guard.git

Set remote (if not set):

    git remote add origin https://github.com/showman-patil/Gear-guard.git

Push main branch:

    git push -u origin main

Notes:
- If you already ran `git remote add origin ...` you can check remotes with `git remote -v`.
- This repo contains the GearGuard Django project (development server: `python manage.py runserver`).
